Iris Machine Learning

von Maximilian Garaycochea

Das Abschlussprojekt verwendet den Iris Datensatz, welcher 150 Iris Pflanzen und Ihre jeweiligen Merkmale (Blütenlänge, -breite etc.) enthält.
Nachdem der Datensatz durchmischt und in Testdaten und Trainingsdaten aufgeteilt wurde, verwendet das Programm zwei verschiedene Modellierungsverfahren
aus scikit-learn, um die Iris Spezien nach Ihren Merkmalen zu Kategorisieren. Es wird das K Nearest Neighbor Modell und das SVM Modell verwendet.
Schließlich werden beide Modelle mithilfe einer 10-fachen Kreuzvalidierung und dem .score Befehl bewertet und es wird eine Vorhersage durchgeführt.

Verwendet wurde der frei verfügbare Iris Datensatz, welcher auf kaggle (https://www.kaggle.com/uciml/iris) runtergeladen werden kann.

Der Programmcode ist unter den Bedingungen der MIT-Lizenz nutzbar.
